Domain: Data Analyst
Type: Beginner

Roadmap:

    Learn the basics of spreadsheets (Excel or Google Sheets: formulas, pivot tables).

    Explore data visualization tools like Tableau or Power BI.

Domain: Data Analyst
Type: Intermediate

Courses:

    "SQL for Data Science" (Coursera)
    "Intermediate Data Visualization with Tableau" (Udemy)

Roadmap:

    Master SQL for querying databases (joins, subqueries, aggregations).
    Deepen knowledge of statistical methods (hypothesis testing, correlation).
    Build advanced visualizations (interactive dashboards, geospatial analysis).
    Learn data cleaning and preprocessing techniques with Python or R.

Docs:

    SQL Documentation (https://dev.mysql.com/doc/)
    Pandas Documentation (https://pandas.pydata.org/docs/)

Learning Platform:

    Mode Analytics
    Databricks Community Edition

Projects Suggestion:

    Query and analyze a large dataset (e.g., e-commerce transactions) using SQL.
    Build an interactive dashboard tracking business KPIs with Tableau.

Certificate Course:

    "IBM Data Analyst Professional Certificate" (Coursera)
    "Data Visualization with Tableau Specialization" (Coursera)

Learning Resources:

    "Practical Statistics for Data Scientists" by Peter Bruce, Andrew Bruce, and Peter Gedeck

Domain: Data Analyst
Type: Advanced

Courses:
